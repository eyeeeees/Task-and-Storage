# 📋 Система учёта задач (Task Management System)

**Разработано в рамках практического задания по дисциплине «Разработка программных приложений»**

---

## 📌 Описание проекта

Приложение предназначено для хранения и управления задачами сотрудников образовательной организации.  
Оно позволяет:
- ✅ Создавать, редактировать и удалять задачи.
- 👤 Назначать исполнителей.
- 📊 Отслеживать статус выполнения.
- 🔍 Искать задачи по названию и фильтровать по статусу.
- 📤 Экспортировать список задач в **PDF** и **Excel**.
- 🌓 Переключать светлую и тёмную тему.
- 🔐 Авторизация и регистрация пользователей с ролями (**Admin / User**).

---

## 🛠️ Технологии

- **Язык программирования:** C# (.NET 8)
- **Графический интерфейс:** WPF (XAML)
- **База данных:** PostgreSQL 16+
- **ORM:** Entity Framework Core 8
- **Библиотеки:**
  - `Npgsql.EntityFrameworkCore.PostgreSQL` — подключение к PostgreSQL
  - `QuestPDF` — экспорт в PDF
  - `EPPlus` — экспорт в Excel
  - `BCrypt` / SHA512 — хеширование паролей (реализовано вручную)

---
