# 📋 Система учёта задач (Task Management System)

**Разработано в рамках производственной практики**

---

## 📌 Описание проекта

Приложение предназначено для хранения и управления задачами сотрудников образовательной организации.  
Оно позволяет:
- ✅ Создавать, редактировать и удалять задачи.
- 👤 Назначать исполнителей.
- 📊 Отслеживать статус выполнения.
- 🔍 Искать задачи по названию и фильтровать по статусу.
- 📤 Экспортировать список задач в **PDF** и **Excel**.
- 🌓 Переключать светлую и тёмную тему.
- 🔐 Авторизация и регистрация пользователей с ролями (**Admin / User**).

---

## 🛠️ Технологии

- **Язык программирования:** C# (.NET 8)
- **Графический интерфейс:** WPF (XAML)
- **База данных:** PostgreSQL 16+
- **ORM:** Entity Framework Core 8
- **Библиотеки:**
  - `Npgsql.EntityFrameworkCore.PostgreSQL` — подключение к PostgreSQL
  - `QuestPDF` — экспорт в PDF
  - `EPPlus` — экспорт в Excel
  - SHA512 — хеширование паролей (реализовано вручную)

## 🚀 Установка и запуск

### 1. Требования
- [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
- [PostgreSQL 16+](https://www.postgresql.org/download/)
- [Visual Studio 2022](https://visualstudio.microsoft.com/ru/) (или любая другая среда для C#)

### 2. Настройка базы данных

Создайте базу данных с именем `tas`:

```sql
CREATE DATABASE tas;
```
В файле `Data/TasDbContext.cs` укажите свою строку подключения:

```csharp
optionsBuilder.UseNpgsql(
    "Host=localhost;Port=5432;Database=tas;Username=postgres;Password=ВАШ_ПАРОЛЬ");
```

Выполните миграции для создания таблиц (в **Package Manager Console** Visual Studio):

```powershell
Add-Migration InitialCreate
Update-Database
```

### 3. Запуск приложения

Откройте решение **`tas.sln`** в **Visual Studio** и нажмите **F5** (или **Ctrl + F5**) для запуска проекта.

## 🧑‍💻 Использование

### 🔐 Регистрация и вход

1. Зарегистрируйте нового пользователя (по умолчанию ему назначается роль **User**).
2. Выполните вход, используя логин и пароль.

### 📝 Управление задачами

- ➕ Нажмите **«Добавить»**, чтобы создать новую задачу.
- ✏️ Выберите задачу и нажмите **«Редактировать»**.
- 🗑️ Для удаления нажмите **«Удалить»** *(доступно владельцу задачи или администратору)*.
- 🔍 Введите текст в поле **«Поиск»** и нажмите **«Найти»**.
- 📂 Выберите статус задачи и нажмите **«Фильтр»**.

### 📤 Экспорт данных

- 📄 **Экспорт PDF** — сохраняет список задач в формате PDF.
- 📊 **Экспорт Excel** — сохраняет список задач в формате Excel.

### 🌓 Смена темы

Нажмите кнопку **«🌓 Тема»**, чтобы переключиться между светлой и тёмной темой оформления.

## 👥 Роли пользователей

| Роль | Возможности |
|------|-------------|
| **Admin** | Полный доступ: создание, экспорт, просмотр, редактирование и удаление всех задач |
| **User** | Создание и экспорт задач |
