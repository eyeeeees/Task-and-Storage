# 📋 Система учёта задач (Task Management System)

**Разработано в рамках практического задания по дисциплине «Разработка программных приложений»**

---

## 📌 Описание проекта

Приложение предназначено для хранения и управления задачами сотрудников образовательной организации.  
Оно позволяет:
- ✅ Создавать, редактировать и удалять задачи.
- 👤 Назначать исполнителей.
- 📊 Отслеживать статус выполнения.
- 🔍 Искать задачи по названию и фильтровать по статусу.
- 📤 Экспортировать список задач в **PDF** и **Excel**.
- 🌓 Переключать светлую и тёмную тему.
- 🔐 Авторизация и регистрация пользователей с ролями (**Admin / User**).

---

## 🛠️ Технологии

- **Язык программирования:** C# (.NET 8)
- **Графический интерфейс:** WPF (XAML)
- **База данных:** PostgreSQL 16+
- **ORM:** Entity Framework Core 8
- **Библиотеки:**
  - `Npgsql.EntityFrameworkCore.PostgreSQL` — подключение к PostgreSQL
  - `QuestPDF` — экспорт в PDF
  - `EPPlus` — экспорт в Excel
  - SHA512 — хеширование паролей (реализовано вручную)

---

## 🚀 Установка и запуск

### 1. Требования
- [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
- [PostgreSQL 16+](https://www.postgresql.org/download/)
- [Visual Studio 2022](https://visualstudio.microsoft.com/ru/) (или любая другая среда для C#)

### 2. Настройка базы данных

Создайте базу данных с именем `tas`:

```sql
CREATE DATABASE tas;
В файле Data/TasDbContext.cs укажите свою строку подключения:

csharp
optionsBuilder.UseNpgsql("Host=localhost;Port=5432;Database=tas;Username=postgres;Password=ВАШ_ПАРОЛЬ");
Выполните миграции для создания таблиц (в консоли диспетчера пакетов Visual Studio):

bash
Add-Migration InitialCreate
Update-Database
3. Запуск приложения
Откройте решение (tas.sln) в Visual Studio и нажмите F5 (или Ctrl+F5) для запуска.

🧑‍💻 Использование
Регистрация — создайте нового пользователя (по умолчанию роль User).

Вход — войдите под своим логином и паролем.

Управление задачами:

Нажмите «Добавить», чтобы создать новую задачу.

Выберите задачу в таблице и нажмите «Редактировать» или «Удалить» (доступно только администратору или владельцу задачи).

Введите текст в поле «Поиск» и нажмите «Найти».

Выберите статус и нажмите «Фильтр».

Экспорт:

Нажмите «Экспорт PDF» или «Экспорт Excel», выберите папку для сохранения.

Тема:

Нажмите «🌓 Тема» для переключения между светлой и тёмной темой.

👥 Роли пользователей
Роль	Права
Admin	Полный доступ: создание, редактирование, удаление всех задач.
User	Создание задач, редактирование/удаление только своих задач.

🧪 Тестирование
В проекте реализованы следующие тест-кейсы:

№	Название теста	Шаги	Ожидаемый результат
1	Успешная регистрация	Заполнить логин/пароль, нажать "Регистрация"	Сообщение "Регистрация успешна"
2	Авторизация	Ввести логин/пароль, нажать "Войти"	Открывается главное окно
3	Добавление задачи	Заполнить все поля, нажать "Сохранить"	Задача появляется в таблице
4	Поиск по названию	Ввести название, нажать "Найти"	Отображаются только подходящие задачи
5	Фильтрация по статусу	Выбрать статус, нажать "Фильтр"	Показываются задачи только с выбранным статусом
6	Удаление задачи	Выбрать задачу, нажать "Удалить", подтвердить	Задача исчезает из списка
7	Экспорт в PDF	Нажать "Экспорт PDF", сохранить файл	Файл создаётся без ошибок
8	Экспорт в Excel	Нажать "Экспорт Excel", сохранить файл	Файл создаётся без ошибок
9	Переключение темы	Нажать "🌓 Тема"	Цвета интерфейса меняются
